# Ledgera Desktop

Desktop application for [Ledgera](https://ledgera.co.in) — the AI compliance management platform for Indian Chartered Accountants.

## Download

Head to the [Releases](https://github.com/Pavanchandar77/Ledgera-desktop/releases) page and download the latest installer for your platform.

| Platform | File |
|----------|------|
| Windows  | `Ledgera-windows.exe` |

## About

This is the Electron wrapper for the Ledgera web application, packaged as a native desktop app with code-signed installers.
